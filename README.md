# 2099 Expense Trakcer
An expense tracker made with django as the backend that helps you track and manage your finances

## How to Run
Clone the repo
```bash
git clone https://github.com/OveaTrint/expense-tracker-django.git
```

Change current directory
```bash
cd expense-tracker-django
```

Install dependencies via uv
```bash
uv sync
```

Run the django server
```bash
python manage.py runserver
```
